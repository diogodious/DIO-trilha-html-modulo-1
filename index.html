Introdução ao HTML

Início

O HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da web.
Ele fornece a estrutura básica para o conteúdo da web, permitindo que os desenvolvedores organizem texto,
imagens e outros elementos de forma hierárquica.

Estrutura Básica de um Documento HTML

Início

Um documento HTML é composto por uma série de elementos aninhados, que são definidos por tags.
A estrutura básica de um documento HTML inclui as seguintes partes:

  1. Declaração do DOCTYPE: Informa ao navegador que tipo de documento está sendo carregado.
  2. Elemento HTML: O elemento raiz que contém todo o conteúdo da página.
  3. Elemento HEAD: Contém informações sobre o documento, como título e links para estilos.
  4. Elemento BODY: Contém o conteúdo visível da página, como texto, imagens e outros elementos.

Elementos e Tags HTML

Início

Os elementos HTML são as unidades básicas de construção de uma página da web.
Cada elemento é definido por uma tag de abertura e uma tag de fechamento,
com o conteúdo do elemento entre elas. Por exemplo:

  • <p>Este é um parágrafo.</p>
  • <h1>Este é um título.</h1> as tags de titulos elas tem niveis que vao do h1 ao h6
  • <img src="imagem.jpg" alt="Descrição da imagem">

As tags HTML podem ter atributos, que fornecem informações adicionais sobre um elemento. Por exemplo:

<a href="https://www.exemplo.com">Este é um link</a>

Atributos HTML

Início

Os atributos HTML são usados para fornecer informações adicionais sobre um elemento. Eles são sempre especificados na tag de abertura e consistem em um nome e um valor, separados por um sinal de igual. Por exemplo:

<a href="https://www.exemplo.com">Este é um link</a>

Alguns atributos comuns incluem:

  • href: Especifica a URL de um link.
  • src: Especifica a URL de uma imagem.
  • alt: Fornece uma descrição alternativa para uma imagem.
  • title: Fornece informações adicionais sobre um elemento (geralmente exibidas como uma dica de ferramenta).
